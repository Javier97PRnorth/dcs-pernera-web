Perneras Digitales

Selecciona un avión para ver su hoja de pernera: cold start básico y procedimientos esenciales.

Aviones

F‑14 Tomcat – Cold Start básico

  • • Baterías ON
  • • Inverters ON
  • • APU START
  • • Engine 1 / 2 START (procedimiento estándar)
  • • Generators ON
  • • HYD PRESS comprobada
  • • INS ALIGN (modo básico, posición aproximada)
  • • Radios y TACAN configurados
  • • Comprobación luces / anti‑colisión
  • • Configuración armamento básica (A/A o A/G)

F‑16 Viper – Cold Start básico

  • • MAIN PWR – BATT
  • • FLCS PWR – ON
  • • JFS START
  • • ENGINE RUN – IDLE estable
  • • GEN – ON
  • • AVIONICS – ON (MMC, FCC, etc.)
  • • INS / GPS ALIGN (modo básico)
  • • Radios, IFF, TACAN / VOR configurados
  • • Configuración armamento (SMS básico)

F‑18C Hornet – Cold Start básico

  • • BATT – ON
  • • APU – ON
  • • ENGINE CRANK – L / R
  • • Generators – ON
  • • INS – ALIGN (modo NAV básico)
  • • Radios, IFF, TACAN configurados
  • • Configuración armamento (MFDs, programas básicos)

A‑10C II – Cold Start básico

  • • Baterías y generadores – ON
  • • APU – START
  • • Engine 1 / 2 – START
  • • CDU / EGI – ALIGN
  • • Radios y TAD configurados
  • • Armamento (DSMS) configurado

MiG‑21bis – Cold Start básico

  • • Baterías – ON
  • • Inverters – ON
  • • Fuel pumps – ON
  • • Engine START
  • • Sistemas eléctricos y de armas – ON
  • • Radios y RSBN / PRMG configurados
