Perneras Digitales

Selecciona un avión para ver su hoja de pernera: cold start básico y procedimientos esenciales, pensados para consulta rápida.

Aviones

F‑14 Tomcat — Cold Start básico

  • Baterías ON
  • Inverters ON
  • APU START
  • Engine 1 START → stable RPM
  • Engine 2 START → stable RPM
  • Generators ON
  • HYD PRESS check
  • INS ALIGN (posición aproximada)
  • Radios y TACAN configurados
  • Lights / anti‑collision check
  • Armamento: selección básica (A/A o A/G)

F‑16 Viper — Cold Start básico

  • MAIN PWR — BATT
  • FLCS PWR — ON
  • JFS START
  • Engine start → idle stable
  • GEN — ON
  • AVIONICS — ON
  • INS / GPS ALIGN (básico)
  • Radios / IFF / TACAN configurados
  • SMS: arma y estaciones básicas

F‑18C Hornet — Cold Start básico

  • BATT — ON
  • APU — START
  • Engine crank L / R
  • Generators — ON
  • INS — ALIGN (NAV básico)
  • Radios / IFF / TACAN configurados
  • MFDs: páginas básicas y armamento

A‑10C II — Cold Start básico

  • Baterías / Generators — ON
  • APU — START
  • Engine 1 / 2 — START
  • CDU / EGI — ALIGN
  • Radios / TAD configurados
  • DSMS: armas y estaciones

MiG‑21bis — Cold Start básico

  • Baterías — ON
  • Inverters — ON
  • Fuel pumps — ON
  • Engine START
  • Sistemas eléctricos y armas — ON
  • Radios / RSBN / PRMG configurados

P‑51D Mustang — Cold Start básico

  • Fuel selector — ON
  • Magnetos / Ignition — ON
  • Primer / Mixture — set
  • Propeller — set
  • Engine start → check oil pressure
  • Radios básicas y transponder
